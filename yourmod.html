Create Your Own Model

To make your own model, please download the zip file named "WTG_User" from GitHub. Please ensure you have Java and Flask installed before running the code.

Running It Natively

  1. Unzip the file.
  2. Open userMachine.py.
  3. Change the user inputs on line 195 (must equal 100 to run properly).
  4. Open the command prompt in the folder.
  5. Type py userMachine.py and wait 5-10 minutes for the code to run.
  6. The visualizations will be saved to the WTG_User folder.

Running It on Browser

  1. Unzip the file.
  2. Open the command prompt in the folder.
  3. Type py app.py and copy the browser link it provides.
  4. Change the numbers to equal 100% and wait 5-10 minutes for the visualization to load.
